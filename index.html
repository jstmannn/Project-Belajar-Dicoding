Apa saja hasil dari yang dipelajari?

Selamat datang di website hasil belajar Dicoding. Website ini dirancang dengan struktur semantic HTML untuk memenuhi kriteria submission pada tugas akhir Dicoding.


Project 1 : Website Artikel

Project pertama adalah membuat website artikel dengan menggunakan HTML dan CSS. Website ini memiliki struktur semantic HTML dan styling menggunakan CSS.


Dokumen HTML ini adalah halaman web yang menyajikan profil Kota Bandung, ibu kota Provinsi Jawa Barat. Di dalamnya terdapat elemen-elemen penting seperti judul, deskripsi, dan navigasi yang mengarahkan pengunjung ke bagian sejarah, geografi, dan wisata.


Konten utama terdiri dari beberapa artikel yang menjelaskan sejarah Bandung, kondisi geografisnya, serta tempat-tempat wisata menarik, dilengkapi dengan gambar untuk memperkaya informasi.


Sidebar menyediakan informasi tambahan, termasuk lambang kota dan tabel data penting seperti hari jadi dan luas total. Halaman ini diakhiri dengan footer yang mencantumkan hak cipta, menjadikannya sumber informasi yang komprehensif dan menarik tentang Kota Bandung.


Project 2 : Landing Page

Dokumen HTML ini adalah landing page untuk Dicoding, sebuah platform pendidikan teknologi. Di bagian head, terdapat pengaturan karakter dan viewport untuk responsivitas, serta judul halaman. Halaman ini juga mengimpor stylesheet CSS, font dari Google Fonts, dan ikon dari Font Awesome untuk meningkatkan tampilan visual.


Header halaman menampilkan logo Dicoding dan menu navigasi yang memudahkan pengunjung untuk mengakses berbagai bagian situs, seperti Beranda, Menu, Apa yang Baru, dan Kontak. Konten utama menjelaskan visi Dicoding dalam mengedukasi talenta digital, disertai dengan gambar yang relevan untuk memperkuat pesan tersebut.


Selain itu, terdapat sidebar yang menampilkan ikon media sosial, yang bertujuan untuk meningkatkan interaksi dan keterlib atan pengguna dengan platform. Secara keseluruhan, halaman ini dirancang untuk memberikan informasi yang menarik dan mudah diakses tentang Dicoding, serta mendorong pengunjung untuk menjelajahi lebih lanjut tentang layanan yang ditawarkan.

Alhamdulillah, saya telah menyelesaikan perjalanan pembelajaran saya di Dicoding Academies! Selama beberapa waktu terakhir, saya telah terlibat dalam proses pembelajaran yang mendalam dan bermanfaat, di mana saya mempelajari dasar-dasar pengembangan web yang sangat penting. Melalui berbagai modul yang disediakan, saya telah menguasai beberapa keterampilan kunci yang akan menjadi fondasi bagi karir saya di dunia teknologi.

Pertama-tama, saya memulai dengan Dasar HTML. Dalam modul "Pengenalan ke HTML", saya belajar tentang struktur dasar HTML dan bagaimana elemen-elemen HTML bekerja sama untuk membangun halaman web yang fungsional. Saya memahami pentingnya penggunaan tag yang tepat dan bagaimana menyusun konten dengan cara yang logis dan terorganisir. Pengetahuan ini memberikan saya fondasi yang kuat untuk menciptakan halaman web yang tidak hanya menarik tetapi juga mudah dinavigasi oleh pengguna.

Selanjutnya, saya mendalami Penerapan Elemen Semantik. Melalui modul "Semantic HTML: Mengorganisasi Halaman Konten", saya belajar bagaimana menggunakan elemen semantik untuk memberikan makna yang lebih dalam pada konten yang saya buat. Saya menyadari bahwa penggunaan elemen semantik tidak hanya membantu dalam pengorganisasian konten, tetapi juga meningkatkan aksesibilitas dan SEO (Search Engine Optimization) halaman web. Dengan menerapkan elemen semantik, saya dapat menciptakan struktur yang lebih jelas dan mudah dipahami, baik oleh pengguna maupun mesin pencari.

Setelah itu, saya beralih ke Styling Menggunakan CSS. Dalam modul "Pengenalan ke Styling", saya belajar cara menerapkan berbagai gaya pada halaman web menggunakan CSS. Saya mengeksplorasi berbagai properti CSS, mulai dari pengaturan warna, font, hingga layout. Saya juga belajar tentang pentingnya responsivitas dalam desain web, yang memungkinkan halaman saya terlihat baik di berbagai perangkat, mulai dari desktop hingga smartphone . Dengan keterampilan ini, saya dapat menciptakan tampilan yang menarik dan profesional, yang tentunya akan meningkatkan pengalaman pengguna.

Terakhir, saya mendalami Layouting Menggunakan Flexbox. Melalui modul "Layout Responsif dengan Flexbox", saya belajar teknik layouting yang modern dan efisien. Flexbox memungkinkan saya untuk mengatur elemen-elemen dalam halaman web dengan cara yang fleksibel dan responsif. Saya memahami bagaimana menggunakan properti Flexbox untuk mengatur posisi dan ukuran elemen dengan mudah, sehingga saya dapat menciptakan desain yang dinamis dan adaptif terhadap berbagai ukuran layar. Keterampilan ini sangat berharga, terutama di era di mana akses web melalui perangkat mobile semakin meningkat.

Dengan semua keterampilan yang telah saya pelajari, saya merasa siap untuk melanjutkan perjalanan saya di dunia pengembangan web. Saya sangat bersemangat untuk menerapkan pengetahuan ini dalam proyek nyata dan terus belajar dari pengalaman. Saya percaya bahwa dengan dedikasi dan semangat yang tinggi, saya dapat terus berkembang dan berkontribusi dalam industri teknologi yang terus berubah ini. Terima kasih Dicoding, atas kesempatan belajar yang berharga ini!
